# AI Engineering Examples

A collection of practical AI and machine learning examples implemented in JavaScript and Python.

## Examples

| Example | Description |
|---------|-------------|
| [example-1-categoria-pessoas](./example-1-categoria-pessoas) | Neural network for classifying people into categories (premium, medium, basic) based on their features |

## Structure

Each example contains:
- `js/` - JavaScript implementation using TensorFlow.js
- `python/` - Python implementation using TensorFlow/Keras
- `README.md` - Detailed explanation of the example

## Getting Started

Navigate to any example folder and follow the instructions in its README to run the code.

## Requirements

- **JavaScript:** Node.js 22
- **Python:** Python 3.9-3.13, pip
